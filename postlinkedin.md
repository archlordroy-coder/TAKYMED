**C'est quoi Takymed ?**
C'est une plateforme intelligente qui connecte patients, pharmaciens et professionnels de santé pour un suivi médical optimal.

**Fonctionnalités clés :**
✅ **Gestion d'ordonnances** : Création et suivi numérique pour ne plus jamais perdre une prescription.
✅ **Rappels de prises intelligents** : Notifications pour garantir le respect des traitements.
✅ **Catalogue de médicaments** : Recherche simplifiée et informations détaillées.
✅ **Gestion de pharmacies** : Outils dédiés pour les pharmaciens (inventaire, clients).
✅ **Calendrier interactif** : Une vue claire sur le parcours de soin.

**Côté technique :**
🛠️ **Frontend** : React 18, Vite, TailwindCSS, TypeScript.
🛠️ **Backend** : Express.js, Node.js.
🛠️ **Base de données** : SQLite (better-sqlite3).
🛠️ **Qualité** : Vitest pour les tests, TypeScript pour un code robuste.
